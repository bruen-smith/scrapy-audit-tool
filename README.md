# scrapy-audit-tool
This project was developed to audit school websites for structural and compliance consistency. It uses Python and multiprocessing to efficiently scan and analyze site components at scale.

## Features
- Automated site scraping
- Parallel processing using multiprocessing
- Data validation checks
- Structured output formatting

## Purpose
This project reflects my interest in systems efficiency and automated problem detection. It was built to explore how scalable tools can identify inconsistencies and improve digital infrastructure.
